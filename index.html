Theoretical Grounding & Core Dichotomies

Before examining the 8 specific cognitive functions, we must define the structural axes of the psyche as derived from Carl Jung's Psychological Types (1921). The four-letter MBTI code is essentially a mathematical shorthand describing the flow of psychic energy (Libido) between the Subject (Self) and the Object (External World).

The Bridge: How J/P Defines the Functions

The J/P dichotomy, introduced by Myers and Briggs, is the master key that translates the 4 letters into the 8 cognitive functions. It answers one vital question: Which cognitive function does the Subject present to the External World (Extravert)?

  • J (Judging): Extraverts their Judging function (Thinking or Feeling). Therefore, an INTJ uses extraverted Thinking (Te). Because their primary interface is Te, their internal dominant function must be a perceiving one (Ni).
  • P (Perceiving): Extraverts their Perceiving function (Sensing or Intuition). Therefore, an INTP uses extraverted Intuition (Ne). Because their primary interface is Ne, their internal dominant function must be a judging one (Ti).

Information Metabolism & The 8 Functions

By combining the dichotomies, we derive the 8 cognitive functions. These represent the specific neurological and psychological pathways through which an individual metabolizes data (Irrational functions: S/N) and organizes that data into decisions (Rational functions: T/F).
